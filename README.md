# El Punto del Sabor

Sitio web estático para un negocio local de comida rápida, con enfoque en una presentación visual atractiva, navegación clara y diseño responsivo.

## Características

- Diseño moderno con estética cálida y branding para comida rápida.
- Secciones clave: portada, menú, horarios, ubicación y pie de página con redes.
- Selector de teléfonos en el encabezado para llamar rápidamente.
- Botones de contacto en el pie de página con estilo moderno.
- Navegación móvil con menú desplegable.

## Estructura del proyecto

```text
punto_sabor/
├─ index.html
├─ README.md
├─ css/
│  └─ style.css
├─ js/
│  └─ app.js
└─ img/
```

## Cómo ejecutar

Este proyecto no requiere build ni dependencias.

1. Abre la carpeta del proyecto en VS Code.
2. Ejecuta `index.html` con una vista previa (por ejemplo, Live Server) o abre el archivo directamente en tu navegador.

## Personalización rápida

- **Textos y secciones:** editar en `index.html`.
- **Estilos y responsive:** ajustar en `css/style.css`.
- **Interacciones (menú y selector de teléfono):** modificar en `js/app.js`.
- **Imágenes:** reemplazar archivos dentro de `img/` manteniendo nombres o actualizando rutas en `index.html`.

## Notas

- El proyecto está preparado para un flujo simple tipo landing page.
- Si deseas escalarlo, se puede migrar fácilmente a un framework sin perder la estructura visual actual.
